PLUSHIE IMAGES DIRECTORY
========================

Place your plushie PNG images in this directory.

NAMING CONVENTION:
------------------
Files should be named: plushie_<color>.png

Where <color> matches the color names used in the game:
- plushie_red.png
- plushie_green.png
- plushie_blue.png
- plushie_lime_green.png  (use underscore for spaces)
- plushie_pink.png
- plushie_white.png
- plushie_orange.png
- plushie_cyan.png
- plushie_purple.png

EXAMPLES:
---------
For a red plushie: plushie_red.png
For a lime green plushie: plushie_lime_green.png

NOTES:
------
- If an image file is missing, the game will fall back to programmatic drawing
- Images are cached in memory after first load for better performance
- Supported formats: PNG (recommended), JPG, GIF
