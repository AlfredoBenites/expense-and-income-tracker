SHOP BACKGROUND IMAGE
=====================

Place your shop background PNG image in this directory (src/main/resources/).

FILENAME:
---------
shop_background.png

REQUIREMENTS:
------------
- Format: PNG (recommended), JPG, or GIF
- The image will be automatically scaled to fit the game window (900x600 pixels)
- If the image is not found, the game will use the programmatic drawing as fallback

LOCATION:
---------
src/main/resources/shop_background.png
